# KBO 도장깨기 - 두근두근 오타니 ⚾🔥

![Game Logo Placeholder](https://img.shields.io/badge/Shohei-Ohtani-blue?style=for-the-badge&logo=baseball)

**"KBO에는 나의 적수가 없구나..."** 

일본과 메이저리그를 평정한 야구 천재 **오타니 쇼헤이**가 KBO 리그에 떴다!
가장 낮은 순위의 구단(키움)부터 시작해 KBO 1위 팀을 꺾고, 마침내 KBO 최고 스타들이 모인 **대한민국 국가대표팀(Team 코리아)**과 최후의 결전을 벌이는 텍스트 기반 야구 미니게임입니다.

---

## 📖 우리의 개발 여정 (The Journey)

이 프로젝트는 단순한 C언어 텍스트 게임에서 시작하여, 수많은 아이디어와 열정이 덧붙여지며 하나의 거대한 패키지 게임으로 진화했습니다.

1. **기반 시스템 구축**: 처음엔 그저 타이밍에 맞춰 타격을 하고 공을 던지는 미니게임이었습니다. 하지만 점차 타격(S), 번트(Space), 도루(R) 등 야구의 디테일한 전술들이 추가되었습니다.
2. **KBO 리그 도장깨기**: 단순한 1회성 게임이 아니라, 2024년 KBO 실제 순위를 반영하여 키움(10위)부터 LG(1위)까지 차례대로 맞붙는 **스테이지 방식**을 도입했습니다.
3. **RPG 요소와 상점 (성장 로직)**: 게임에서 승리(3000원)하거나 패배(1000원)하여 얻은 자금으로 **대기실 상점**에서 오타니의 스탯(근력, 민첩, 지구력)을 육성할 수 있게 만들었습니다. 어려운 팀을 만나면 패배의 쓴맛을 보며 스펙업을 하는 '로그라이크'적인 재미를 잡았습니다.
4. **리얼리티 추가 (날씨와 피로도)**: 매 경기 무작위로 **맑음/비/눈** 날씨가 적용되어 난이도에 변수를 주고, 이닝이 길어질수록 **피로도(CC)**가 쌓여 스탯에 페널티를 주는 시스템으로 긴장감을 높였습니다. 수비 시에는 직접 WASD로 낙구 지점을 향해 뛰어가는 미니게임까지 챙겼습니다!
5. **완벽한 디테일과 엔딩 연출**: 각 팀의 상징색에 맞춘 도트 모자 색상부터, 홈구장 이름 출력, 그리고 대망의 1위 팀을 꺾었을 때 나타나는 **소년 만화풍의 숨겨진 보스(Team 코리아) 난입 컷신**까지! 한 편의 스포츠 만화를 보는 듯한 연출을 완성했습니다.

## 🎮 주요 기능 (Features)

* **다이나믹한 타격 및 투구**: 타석에서는 투구 속도와 타이밍을 계산해 스윙/번트/도루를 선택하고, 마운드에서는 빠르게 움직이는 게이지를 맞춰 포크볼/커브볼/직구를 던집니다.
* **스테이지 난이도 스케일링**: 상위 스테이지(팀)로 올라갈수록 투수의 구속이 눈에 띄게 빨라지며, AI 타자들의 타격 확률과 반응 속도가 극적으로 상승합니다.
* **스탯 시스템**:
  * 💪 **근력**: 장타(2루타, 3루타, 홈런) 확률 증가
  * ⚡ **민첩**: 도루 성공률 증가 및 수비 시 이동 속도 상승
  * 🛡️ **지구력**: 후반 이닝에서 피로도 페널티를 방어
* **더블 스틸(이중 도루)**: 주자가 1, 2루에 있을 때 도루를 시도하면 이중 도루가 발동하는 등 리얼한 야구 룰을 구현했습니다.

## 🛠️ 조작 방법 (Controls)

* **로비/메뉴**: `방향키(↑, ↓)` 로 이동, `Space` 또는 `Enter` 로 선택
* **타격 (공격)**:
  * `S`: 타이밍에 맞춰 스윙
  * `Space`: 기습 번트 (50% 확률로 1루 진루 혹은 아웃)
  * `R`: 도루 시도 (투수가 공을 던지기 전 준비 시간에 입력)
* **투구 (수비)**: `←` (커브), `→` (포크볼), 그 외 방향키 (직구) 선택 후 타이밍 게이지 맞추기
* **수비 (외야 플라이)**: `W`, `A`, `S`, `D` 또는 `방향키`를 이용해 제한 시간 내에 낙구 지점(`X`)으로 이동

## 🚀 빌드 및 실행 방법

1. 저장소를 클론합니다.
   ```bash
   git clone https://github.com/Junho73/Otani-Baseballgame.git
   ```
2. **Visual Studio**에서 `Iloveotani.slnx` (또는 `.vcxproj`) 솔루션 파일을 엽니다.
3. 상단 메뉴에서 **빌드 -> 솔루션 다시 빌드 (Ctrl+Alt+F7)** 를 실행합니다.
4. **F5**를 눌러 "두근두근 오타니"의 장대한 KBO 정복기를 시작하세요!
